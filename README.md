## Project Description

This project is an improved version of my Assignment 1 portfolio website. It keeps the original structure and design, then adds more interactivity and a better user experience using JavaScript, dynamic content, animations, and user feedback.

The website includes:

- About section with a personalized greeting
- Projects section with live search and category filtering
- Contact form with validation and feedback messages
- Dark / light theme toggle with saved preference
- Scroll reveal animations and hover transitions

## Setup Instructions

### Option A: Run Locally

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

### Option B: VS Code Live Server

1. Open the project folder in **VS Code**.
2. Install the **Live Server** extension if needed.
3. Right-click `index.html`.
4. Select **Open with Live Server**.

## Features Implemented

### Dynamic Content
- Live project search while typing
- Project category filter buttons
- Personalized greeting that changes based on time of day and saved visitor name

### Data Handling
- `localStorage` stores:
  - Theme preference
  - Visitor name
- Contact form validates user input and shows confirmation or error messages

### Animation and Transitions
- Smooth hover effects on cards, buttons, and navigation links
- Fade-in reveal effect when sections enter the screen
- Animated form success and error messages

### Error Handling and User Feedback
- Clear error messages for invalid form input
- Empty-state message when no projects match the search/filter
- Friendly confirmation message when a name is saved
